Project Overview
The Petshop is an industrial-grade e-commerce storefront designed for efficiency and high performance. It features a responsive, glassmorphism-based UI, client-side state management, and an integrated "Webfluid AI" assistant for enhanced user interaction.

Core Features
Reactive Shopping Cart: Real-time state management ensuring a seamless "add-to-cart" and checkout experience.

Dynamic Search Architecture: Instant product filtering and sorting with real-time UI updates.

Webfluid AI (BETA): Integrated local AI sidebar for intelligent user assistance and navigation queries.

Modern UI/UX: High-density, glassmorphism-inspired design with a fluid, dark-mode color palette.

API-Ready Checkout: Modular logic prepared for seamless backend integration (Cash on Delivery enabled).

Tech Stack
Frontend: HTML5, CSS3 (Custom Glassmorphism), and Vanilla JavaScript.

Data Handling: Reactive DOM manipulation and browser-based state management.

Design System: Custom utility-first CSS architecture for high performance.
